# Roadmap

## Vision
A C++/OpenGL-based 3D editor and rendering sandbox focused on learning graphics, tooling, and later mechanical simulation.

## Current state
- Scene hierarchy
- Transform system
- Viewport rendering
- Selection / picking
- YAML serialization
- Material / shader system
- Basic lighting
- Particle experiments

## North Star
- Mini editor with modern rendering features
- Mechanical sandbox foundation with game mode support
- Lua ​​scripting entry point
- Well-documented architecture
