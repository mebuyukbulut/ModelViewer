# 0003 - Keep Project Docs Inside Repo

## Status
Accepted

## Context
Projenin büyümesiyle beraber dökümantasyon gerekçesi ortaya çıkmıştır. Bu dökümantasyonun saklanması için, notion, blog veya repo kullanılabilir. 

## Decision
Projenin dökümantasyonu projenin reposunda bulunacak.

## Alternatives considered
- Notion vb. not sistemi
- Blog (internet sitesi)

## Rationale
Diğer alternatifler görsel olarak daha iyi bir sonuç verse de sürdürülebilirlik ve güncellik açısından en iyi seçenek repo. 

## Consequences
### Positive
- .md dosyaları kullanacağımız için kullanımı nispeten basit ve sade
- repoda durduğu için güncel tutması kolay.

### Negative
- Görsel açıdan diğer seçeneklere göre daha kötü
