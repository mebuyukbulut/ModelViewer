# 0013 - Mermaid 

# Durum 
Dökümantasyonda şemaları gösterecek bir araça ihtiyacımız var. 

# Karar
Mermaid kullanmak. Sebepleri:
- Markdown içine gömebiliyoruz. 
- Resim vb alternatiflere göre daha pratik. 
- Şuanki tüm şema ihtiyacımızı karşılayacak nitelikte. 

# Alternatifler
- Resim kullanmak: Güzel fakat daha fazla efor istiyor.
- Mermaid alternatifi bir yapı kullanmak: Mümkün fakat ihtiyaçlarımı karşılayan bir tool buldum ve daha fazla araştırmayı gereksiz görüyorum. 


# Pozitif Çıktılar
