## Philosophy

MokaLab is not intended to be a full production engine. It is a graphics laboratory for exploring rendering techniques, tool architecture and engine programming ideas.

## Features

- Physically Based Rendering (PBR)
- Shadow Mapping
- Render Pass Architecture
- Material System
- Asset Management System

## Roadmap

- [x] PBR pipeline
